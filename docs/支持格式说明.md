# PaddleOCR Web UI 支持格式说明

## 直接 OCR 识别（PaddleOCR 原生支持）

以下格式由 PaddleOCR 直接处理，无需转换：

| 扩展名 | 中文名称 | 说明 |
|--------|---------|------|
| pdf | PDF 文档 | 便携式文档格式，逐页转图片后 OCR |
| jpg / jpeg | JPEG 图片 | 联合图像专家组格式，常见照片/扫描件 |
| png | PNG 图片 | 便携式网络图形，支持透明通道 |
| bmp | BMP 图片 | 位图格式，无压缩 |
| tiff / tif | TIFF 图片 | 标签图像文件格式，常用于扫描/传真 |
| webp | WebP 图片 | 谷歌开发的现代图片格式，体积小 |

## LibreOffice 转 PDF 后 OCR 识别

以下格式通过 LibreOffice headless 转为 PDF，再由 PaddleOCR 识别：

| 扩展名 | 中文名称 | 说明 |
|--------|---------|------|
| doc | Word 文档（旧版） | Microsoft Word 97-2003 二进制格式，**必须安装 LibreOffice** |
| docx | Word 文档 | Microsoft Word 2007+ XML 格式 |
| odt | OpenDocument 文本 | LibreOffice/OpenOffice 文字处理格式 |
| rtf | 富文本格式 | 通用富文本交换格式 |
| xls | Excel 表格（旧版） | Microsoft Excel 97-2003 二进制格式，**必须安装 LibreOffice** |
| xlsx | Excel 表格 | Microsoft Excel 2007+ XML 格式 |
| ods | OpenDocument 电子表格 | LibreOffice/OpenOffice 表格格式 |
| csv | CSV 表格 | 逗号分隔值，纯文本表格 |
| ppt | PowerPoint 演示文稿（旧版） | Microsoft PowerPoint 97-2003 二进制格式 |
| pptx | PowerPoint 演示文稿 | Microsoft PowerPoint 2007+ XML 格式 |
| odp | OpenDocument 演示文稿 | LibreOffice/OpenOffice 演示文稿格式 |
| txt | 纯文本 | 普通文本文件 |
| html / htm | HTML 网页 | 超文本标记语言 |

## 处理流程

```
图片文件（jpg/png/bmp/tiff/webp）
    ↓ 直接 OCR
    PaddleOCR 识别

PDF 文件（pdf）
    ↓ 逐页转图片
    PaddleOCR 识别

Office 文件（doc/docx/xls/xlsx/ppt/pptx/odt/ods/odp/rtf/csv/txt/html）
    ↓ LibreOffice headless 转 PDF
    ↓ 逐页转图片
    PaddleOCR 识别

CAD 文件（dwg/dxf）
    ↓ cad2x 转 PDF
    ↓ 逐页转图片
    PaddleOCR 识别
```

## 无 LibreOffice 时的降级方案

当 LibreOffice 不可用时，docx 和 xlsx 格式可通过 Python 库直接提取文本：

| 扩展名 | 降级方式 | 使用库 |
|--------|---------|--------|
| docx | python-docx 提取文本 | python-docx |
| xlsx | openpyxl 提取文本 | openpyxl |
| doc | **无法处理** | 需 LibreOffice |
| xls | **无法处理** | 需 LibreOffice |

## cad2x 转 PDF 后 OCR 识别

以下 CAD 格式通过 cad2x 工具转为 PDF，再由 PaddleOCR 识别：

| 扩展名 | 中文名称 | 说明 |
|--------|---------|------|
| dwg | AutoCAD 图纸 | AutoCAD 原生二进制格式（R2.5~R2018） |
| dxf | DXF 交换文件 | AutoCAD 文本交换格式 |

## 共计支持格式：24 种

- 直接 OCR：7 种（pdf, jpg, jpeg, png, bmp, tiff, tif, webp）
- LibreOffice 转换：15 种（doc, docx, odt, rtf, xls, xlsx, ods, csv, ppt, pptx, odp, txt, html, htm）
- cad2x 转换：2 种（dwg, dxf）
